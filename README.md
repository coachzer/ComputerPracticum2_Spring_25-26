# Computer Practicum 2 EN/SLO

This repository contains the materials for the Computer Practicum 2 course, offered in English and Slovenian.

Class start date: in the week of February 16, 2026

## Course Overview

The Computer Practicum 2 course is designed to provide students with practical experience in HTML, CSS, and JavaScript. The course covers the following topics:

- HTML structure and semantics
- CSS styling and layout techniques
- JavaScript programming fundamentals

## Implementation Plan

The course is structured into weekly modules.

All the content refers to titles of [W3Schools](https://www.w3schools.com/) tutorials.

For additional resources, check out the [MDN Curriculum](https://developer.mozilla.org/en-US/curriculum/) and their modules.
